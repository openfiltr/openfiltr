# Roadmap

## v1.0 - Foundation

- [x] DNS forwarding
- [x] Block rules (exact, wildcard, regex)
- [ ] Allow-rule precedence
- [x] Rule sources (hosts format)
- [x] Local DNS entries (A, AAAA, CNAME)
- [ ] Per-client / per-group policies
- [x] REST API + OpenAPI 3.1 spec
- [x] YAML import & export
- [x] Docker & curl install
- [x] Embedded bbolt persistence
- [x] Optional PostgreSQL persistence
- [x] Local user auth + API tokens
- [x] Activity log & audit trail
- [x] Configuration export & import
- [ ] Admin web UI foundation

## v1.1 - DNS Hardening & Automation

- [ ] DNSSEC validation
- [ ] DoH upstream support
- [ ] DoT upstream support
- [ ] Webhook events for rule changes
- [ ] Client tags
- [ ] Role-based access control
- [ ] Prometheus metrics endpoint

## v1.2 - Extensibility

- [ ] Plugin system
- [ ] Theme customisation
- [ ] SSO integration
- [ ] Policy templates

## Influencing the roadmap

Open an RFC issue to propose significant changes. The community votes via 👍 reactions and comments.
